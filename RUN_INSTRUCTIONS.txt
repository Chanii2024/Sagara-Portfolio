🚀 SAGARA PORTFOLIO - SETUP INSTRUCTIONS

Follow these steps to get the portfolio up and running smoothly:

1. INITIAL SETUP:
   Ensure you have Node.js and MongoDB installed on your system.
   Open your terminal in the project root folder.

2. INSTALL DEPENDENCIES:
   If this is the first time running the project, run:
   npm install

3. DATABASE CONFIGURATION:
   - Start your MongoDB service (default port: 27017).
   - The app will automatically create the 'sagara_portfolio' database.

4. START THE SERVER:
   Execute the following command:
   npm start

5. ACCESS THE APPLICATION:
   - Portfolio: http://localhost:3000
   - Admin Panel: http://localhost:3000/admin.html

6. ADMIN CREDENTIALS:
   - Username: sagara
   - Password: 123
