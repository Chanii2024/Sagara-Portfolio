# HOW TO RUN THE WEBSITE FAST

1. OPEN TERMINAL:
   Make sure you are in the project root folder:
   c:\Work - Chanii\Projects\FrameMe

2. START THE SERVER:
   Run the following command:
   npm start

3. ACCESS THE WEBSITE:
   Open your browser and type:
   http://localhost:3000

---
NOTE: 
- Make sure MongoDB is running on your system (default port 27017).
- The website might take a few seconds to load the 1,600 animation frames.
